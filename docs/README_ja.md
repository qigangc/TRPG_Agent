# TRPG Agent テキストTRPG AI

智譜AI（GLM-4）搭載のテキストベースTRPGエージェント。GradioウェブUI付き。2つの異なる世界観——**DND**と**CNC**——をサポートし、それぞれ独自の叙事スタイルと専用メカニクスを備えています。

[English](README_en.md) | [简体中文](README_zh.md) | [繁體中文](README_zh_TW.md) | [日本語](README_ja.md)

## 機能

- **デュアル世界観切替** — DND（叙事ファンタジー）とCNC（コメディ中国ファンタジー）をいつでも切替可能、キャラクターデータは保持
- **AIゲームマスター** — 智譜AI GLM-4がDM/GMを務め、ストリーミング応答に対応
- **D20判定システム** — AIが判定を要求した際に自動でダイスロール、クリティカル成功/失敗を処理
- **キャラクター成長** — 6属性、経験値、レベルアップ、属性ポイント配分
- **プリセットキャラクター** — 5つの即用キャラクターテンプレート（戦士、ローグ、メイジ、バード、モンク）、初心者でもすぐ始められる
- **世界観専用メカニクス** — DNDにはインスピレーションダイス；CNCには修仙ブレイクスルー
- **セーブ＆ロード** — JSONベースのセーブファイル、無制限スロット

## クイックスタート

```bash
cd TRPG_Agent
pip install -r requirements.txt
cp .env.example .env       # .envを編集、ZHIPU_API_KEYを入力
python main.py             # http://localhost:7860 を開く
```

## ゲームの流れ

1. **世界観を選択** — メインページでDNDまたはCNCを選択
2. **セーブロードまたはキャラクター作成** — 既存のセーブをロード、または新キャラクターを作成
3. **冒険開始** — AIゲームマスターと対話、ダイスロール、キャラクター成長

## ルール

### ダイス

d20のみ使用。ダメージダイスなし——戦闘結果はAIの叙事で決定。

**判定公式：** `d20 + 属性修正値 ≥ DC`

| DC | 難易度 |
|---|---|
| 10 | 簡単 |
| 15 | 普通 |
| 20 | 困難 |
| 25 | 極難 |

ナチュラル20 = クリティカル成功（必ず成功）。ナチュラル1 = クリティカル失敗（必ず失敗）。

### キャラクター属性

6属性のデフォルト値は10、修正値 = `(値 - 10) // 2`：

| 属性 | 修正値（値14の場合） |
|---|---|
| 筋力（力量） | +2 |
| 敏捷力（敏捷） | +2 |
| 体力（体质） | +2 |
| 知力（智力） | +2 |
| 判断力（感知） | +2 |
| 魅力（魅力） | +2 |

**キャラクター作成：** 20ポイントの属性ポイントを配分（10以上1ポイントにつき1消費）。

**成長システム：** 経験値閾値 = `レベル × 100`。レベルアップで+1属性ポイント。

## 設定

すべての設定は `.env` ファイルまたは `config.py` のデフォルト値で構成可能：

| 変数 | デフォルト | 説明 |
|---|---|---|
| `ZHIPU_API_KEY` | — | **必須。** 智譜AI APIキー |
| `MODEL_NAME` | `glm-4` | 智譜AIモデル名 |
| `MAX_HISTORY` | `20` | コンテキストに保持する会話ターン数 |
| `INITIAL_ATTRIBUTE_POINTS` | `20` | キャラクター作成時の属性ポイント |
| `TEMPERATURE` | `0.85` | LLMサンプリング温度 |
| `MAX_TOKENS` | `2048` | 1回の応答あたりの最大トークン数 |
| `MAX_RETRIES` | `3` | LLM APIリトライ回数 |
